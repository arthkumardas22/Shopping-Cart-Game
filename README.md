# Shopping Cart Game

A simple Java console application that allows users to enter an item, its price, and quantity, then calculates the total cost.

## Features

* User input using Scanner
* Total cost calculation
* Simple and beginner-friendly

## Run

```bash
javac ShoppingCartGame.java
java ShoppingCartGame
```

## Example

```text
What would you like to Buy? Apple
What is the price for Each item? 10
How many would you like? 5

You have bought 5 Apple/s
Your total is $50.0
```
